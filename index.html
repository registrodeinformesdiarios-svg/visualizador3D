Protocolo file:// bloqueado
El navegador bloquea las peticiones a Google Drive cuando el archivo se abre directamente desde el disco. Es necesario correr un servidor local.
Solución rápida — ejecuta este comando en la carpeta del proyecto:
python -m http.server 8080
Luego abre: http://localhost:8080
Pasos en Windows
1 Abre el Explorador de archivos en la carpeta donde está el index6.html
2 Escribe cmd en la barra de dirección y presiona Enter
3 Pega el comando de arriba y presiona Enter
4 Abre Chrome y ve a http://localhost:8080
